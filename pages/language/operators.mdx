# Operators
The following operators are supported in pql today.
 - [as](operators/as)
 - [count](operators/count)
 - [extend](operators/extend)
 - [join](operators/join)
 - [project](operators/project)
 - [sort](operators/sort)
 - [summarize](operators/summarize)
 - [take/limit](operators/take)
 - [top](operators/top)
 - [where/filter](operators/where)
