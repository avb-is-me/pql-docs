# Scalar Functions
The following scalar functions are supported in pql today.
 - [count](scalar-functions/count)
 - [countif](scalar-functions/countif)
 - [iff](scalar-functions/iff)
 - [isnotnull](scalar-functions/isnotnull)
 - [isnull](scalar-functions/isnull)
 - [not](scalar-functions/not)
 - [now](scalar-functions/now)
 - [strcat](scalar-functions/strcat)
 - [tolower](scalar-functions/tolower)
 - [toupper](scalar-functions/toupper)

## Notice
Much of the documentation for these scalar-functions was taken from Microsoft' official
[Kusto Query Language repository.](https://github.com/microsoft/Kusto-Query-Language) which is licensed
under Apache 2.0.
